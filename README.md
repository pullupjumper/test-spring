# test-spring
git status，查询仓库状态  
git init，初始化仓库  
git add，添加文件  
git commit，提交文件  
git log，查询提交日志  
git branch，拉分支  
git checkout，切换分支或者标签  
git merge，合并分支  
git branch -d & git branch -D，删除或者强制删除分支  
git tag，添加标签  
# 操作  
1. 關聯遠程倉庫
  + git remote add origin https://github.com/guobinhit/springmvc-tutorial.git  
2. 同步远程仓库和本地仓库  
  + git pull origin master  
3. 将本地仓库修改（或者添加）的内容提交到远程仓库
  + git push origin master
